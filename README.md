## About

Explores Dependency Injection (DI) in Spring Boot, a core principle of the Spring Framework. This assignment covers IoC container mechanics, bean lifecycle, and various injection strategies.

## Key Concepts

- Inversion of Control (IoC) and DI principles
- `@Component`, `@Service`, `@Repository` stereotypes
- Constructor, setter, and field injection
- Bean scopes and lifecycle management
